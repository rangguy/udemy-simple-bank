Catatan untuk mengeksekusi SQL di project simple-bank

1. Jalankan PostgreSQL dengan Docker
   make postgres

2. Buat database simple_bank
   make createdb

3. Terapkan migrasi SQL ke database
   make migrateup

4. Jika perlu membatalkan satu langkah migrasi terakhir
   make migratedown

5. Jika perlu membatalkan semua migrasi
   make migratedownall

6. Generate kode SQL dari query dengan sqlc
   make sqlc

7. Jalankan test untuk memastikan semuanya bekerja
   make test

8. Menggunakan golang-migrate
   dokumentasinya ada di sini https://github.com/golang-migrate/migrate
   migrate create -ext sql -dir (db/migration / path) -seq (init_schema / nama file)

Keterangan:
- File migrasi SQL ada di `db/migration/`
- Query SQL berada di `db/query/`
- Konfigurasi sqlc ada di `sqlc.yaml`
- Database PostgreSQL dapat diakses di `localhost:5433`
- Username: `root`, password: `secret`
